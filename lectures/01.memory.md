#### Куча (Heap)
```
int* i = (int*) malloc(sizeof(int));
std::string* name = new std::string();
char* data = new char[5];
...
free(i);
delete(name);
delete[] data;
```

#### Data segment
```
static const int i = 5;
static std::string name;
extern char data[5];
```
